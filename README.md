# 📝 Lista de Tarefas

Aplicação web simples de lista de tarefas desenvolvida com HTML, CSS e JavaScript, permitindo adicionar e remover tarefas com persistência de dados no navegador.

---

## 🚀 Funcionalidades

- Adicionar novas tarefas
- Remover tarefas da lista
- Salvamento automático das tarefas (localStorage)
- Interface simples e responsiva

---

## 🛠 Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript (DOM e localStorage)

---

## 💻 Como executar o projeto

1. Baixe ou clone este repositório
2. Abra o arquivo `index.html` no navegador
3. Comece a adicionar suas tarefas
